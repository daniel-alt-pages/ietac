# EduManager - Portal de Gestión Académica

Un sistema moderno para le gestión de estudiantes construido con Next.js y Tailwind CSS.

## Tecnologías
- **Framework**: Next.js 15+ (App Router)
- **Estilos**: Tailwind CSS
- **Iconos**: Lucide React
- **Animaciones**: Framer Motion

## Instrucciones de Despliegue en Vercel

1.  Asegúrate de tener una cuenta en [Vercel](https://vercel.com).
2.  Instala Vercel CLI si no lo tienes:
    ```bash
    npm i -g vercel
    ```
3.  Ejecuta el comando de despliegue en esta carpeta:
    ```bash
    vercel
    ```
4.  Sigue las instrucciones en pantalla (Log in, Link to Project, etc). Usa los valores por defecto.
5.  Para despliegue a producción:
    ```bash
    vercel --prod
    ```

## Desarrollo Local
1.  Instalar dependencias:
    ```bash
    npm install
    ```
2.  Correr servidor de desarrollo:
    ```bash
    npm run dev
    ```
